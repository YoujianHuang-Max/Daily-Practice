# Data Engineering Learning Journal

This repository documents daily learning and hands-on practice focused on 
connecting data analyst skills with data engineering concepts and tools.

## Purpose
- Bridge data analysis knowledge with data engineering workflows
- Practice real-world tools step by step
- Build a clear learning trail that shows growth over time

## Tools & Technologies
- Python
- SQL (PostgreSQL)
- Apache Airflow
- Docker
- Git & GitHub

## Learning Structure

### Day 1 – Data Foundations
- Data cleaning and basic exploration
- Understanding datasets from an analyst perspective

### Day 2 – Workflow Orchestration (Airflow)
- Dockerized Airflow setup
- Creating and running first DAG
- Understanding scheduler, webserver, and DAG lifecycle

### Day 3 – SQL Refresh
- Table design: patients, encounters, lab_results
- JOIN, aggregation, window functions
- Analytical queries for healthcare-style data
